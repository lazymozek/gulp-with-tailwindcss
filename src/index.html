Usage

Use the following commands to get started.

// To Install Dev Depedencies
npm install

// To start development and server for live preview
npm run dev

// To generate minifed,optamized files for production server
npm run prod

Options

To change the path of files and destination/build folder configure at config.js file

config: { ... port: 9050 //Update port here }, paths: { root: "./", src: { base: "./src", css: "./src/css", js: "./src/js", img: "./src/img" }, dist: { base: "./dist", css: "./dist/css", js: "./dist/js", img: "./dist/img" }, build: { base: "./build", css: "./build/css", js: "./build/js", img: "./build/img" } }
